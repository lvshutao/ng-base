DecimalPipe - 数字

number_expression | number [:digitInfo[:locale]]
digitInfo => miniIntegerDigits.minFractionDigits-maxFractionDigits
miniIntegerDigits:整数部分最小保留位数，默认为 1
minFractionDigits:分数部分最小保留位数，默认为 0
maxFractionDigits:分数部分最大保留位数, 默认为 3
locale
默认 {{e}} => {{e | number}}
number:'3.5-5' => {{e | number:'3.5-5'}}
number:'3.3-8' => {{e | number:'3.3-8'}}

PercentPipe — 百分比

number_expression | percent[:digitInfo[:locale]]
digitInfo 参考 DecimalPipe
locale
默认 {{p}} => {{ p | percent }}
percent:'3.3-5' => {{p | percent:'3.3-5'}}

CurrencyPipe - 货币

number_expression | currency[:currencyCode[:display[:digitInfo[:locale]]]]
currencyCode,货币符号，人民币 CNY，香港 HKD
display
code: 默认为 USD
symbol: 符号，例如 $
symbol-narrow: 某些国家有两个符号
digitInfo: 参考 DecimalPipe
local
  1. 默认 {{ money }} => {{ money | currency }}
  2. currency:'CNY': 'symbol':'4.1-4' => {{ money | currency:'CNY': 'symbol':'4.1-4'}}
  3. currency:'HKD': 'symbol':'4.1-4' => {{ money | currency:'HKD': 'symbol':'4.1-4'}}
